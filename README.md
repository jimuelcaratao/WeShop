# WeShop

---

## Step to run

### Locally

-   create db in phpmyadmin
-   composer install
-   weshopdb (dbname)
-   npm install
-   npm run dev
-   php artisan storage:link
-   php artisan migrate

## Packages

-   Jetstream - https://jetstream.laravel.com/2.x/introduction.html
-   Socialite - composer require laravel/socialite
